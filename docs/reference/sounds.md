# Sounds Reference

Complete list of all sound effect IDs available in Hytale. Use these IDs when playing sounds via the API.

## Usage

```java
// Play a sound at a location
world.playSound("SFX_Player_Jump", location);

// Play a sound for a specific player
player.playSound("SFX_Chest_Wooden_Open");
```
