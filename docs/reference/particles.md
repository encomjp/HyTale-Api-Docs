# Particle Types

Common Particle Types for `world.spawnParticle()`.

| ID | Description |
| :--- | :--- |
| `FLAME` | Small fire flame. |
| `SMOKE` | Generic smoke. |
| `EXPLOSION` | Large explosion cloud. |
| `HEART` | Breeding/Healing hearts. |
| `BUBBLE` | Water bubbles. |
| `FIREWORK` | Sparkles. |
| `DRIP_WATER` | Ceiling drips. |
| `DRIP_LAVA` | Ceiling lava drips. |
| `NOTE` | Music note. |
| `CRIT` | Critical hit spark. |

> [!NOTE]
> **Custom Particles**
> Hytale allows creating entirely new particle effects in the `assets/particles/` folder using a JSON definition. You are not limited to this hardcoded list!
