# Data Reference

This section lists common IDs and keys used in Hytale development.

## Namespaced Keys
Hytale uses **Namespaced Keys** (e.g., `hytale:stone`) to identify content.
- **Namespace**: The mod or author (e.g. `hytale`, `my_mod`).
- **Path**: The specific ID (e.g. `stone`, `skeleton_warrior`).

## Categories
- [Blocks](./blocks): Terrain and building blocks.
- [Entities](./entities): Mobs, animals, and monsters.
- [Items](./items): Weapons, tools, and consumables.
- [Sounds](./sounds): Audio events and categories.
- [Particles](./particles): Visual effects.
